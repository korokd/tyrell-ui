Quando se é um colaborador podemos fazer orcamentos de pedidos. Já quando estamos logados como Admins podemos criar modelos, opcionais e tambem fazer orcamentos

A criação desta aplicação se deve ao módulo 6 do projeto crescer, na CWI Software.
Onde os alunos foram instruidos a utilização dessas tecnologia e foram requisitados a criar um sistema baseado nesses modos
